- **Travel Agency Managment System** – PHP Laravel , Nginx, CI/CD, Netdata Monitoring, and Discord real-time logging.
- **NGO Project CRM** – PHP Laravel, GitHub Actions CI/CD, Dockerized apps, Nginx + SSL, central logging.
- **Medical Insurance CRM** – PHP Laravel, Jenkins pipelines, backups, monitoring, hardened hosts.
